# glace

![demo](helmet.gif)

This is a toy renderer using rust to learn wgpu and computer graphics built with bevy.

It can also be used to showcase integrating a custom renderer to bevy.

## Features

* Basic Blinn-Phong shading
* Normal mapping
* Specular mapping
* Instanced rendering
* Load obj
* Load gltf
* egui integration
* 3d camera controller
